Unlock Full Power with a Paid Key

CareerForge works best with a paid Google Cloud Project. This removes rate limits and ensures the app never crashes during heavy generation tasks. Don't worry—personal usage typically costs less than $1/month.

1

Create a Project

Go to Google AI Studio and sign in with your Google account.

Visit AI Studio
2

Set up Billing

Crucial Step: To access the paid tier, you must link a billing account (credit card) to your project in the Google Cloud Console.

3

Generate API Key

Click "Create API Key" and select your new paid project.

4

Enter into App

Paste the key starting with AIza... into CareerForge. Check "Remember Key" to save it securely.

Note: While there is a free tier, it has strict rate limits (RPM) that may cause the app to fail when generating complex kits.
